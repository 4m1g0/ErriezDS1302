7 ![DS1302 RTC](https://raw.githubusercontent.com/Erriez/DS1302/master/extras/DS1302.png)

9 ## Library features
10 
11 - Read / write RTC date and time.
12 - Read / write 31 Bytes battery backupped RTC RAM.
13 - Optimized IO interface for AVR targets (Maximum 169kHz CLK Arduino UNO with 16MHz XTAL).
14 
15 ## DS1302 specifications
16 
17 **IMPORTANT NOTES:**
18 
19 * The DS1302 RTC time may deviate up to 1 minute each day, so this device is not recommended for designs with high precision requirements.
20 * Use the high precision DS3231 I2C RTC instead for new designs.
21 * The 3-wire interface is **NOT** compatible with SPI.

42 ## Usage
43 
44 **Initialization**
45 
46 ```C++
47 #include <DS1302.h>
48 
49 // Connect DS1302 data pin to Arduino DIGITAL pin
50 #define DS1302_CLK_PIN 2
51 #define DS1302_IO_PIN 3
52 #define DS1302_CE_PIN 4
53 
54 // Create DS1302 RTC object
55 DS1302 rtc = DS1302(DS1302_CLK_PIN, DS1302_IO_PIN, DS1302_CE_PIN);
56 
57 void setup()
58 {
59  // Initialize RTC
60  rtc.begin();
61 
62  // Make clock and RAM registers writable
63  rtc.writeProtect(false);
64 
65  // Enable RTC clock
66  rtc.halt(false);
67 }
68 ```
